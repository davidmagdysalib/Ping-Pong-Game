# 🧱 Bricks Falling Game

A simple arcade-style game built with Python and the Turtle graphics module. Your objective is to dodge the falling bricks for as long as possible while your score continues to increase. As the game progresses, the challenge becomes more difficult, testing your reflexes and timing.

## 🎮 Features
- Randomly generated falling bricks
- Smooth player movement
- Collision detection
- Live score tracking
- Increasing difficulty over time
- Clean and minimal design

## 🛠️ Built With
- Python 3
- Turtle Graphics

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/davidmagdysalib/Bricks-Falling-Game.git
   ```
2. Open the project folder:
   ```bash
   cd Bricks-Falling-Game
   ```
3. Run the game:
   ```bash
   python main.py
   ```

## 🎯 Controls
- ⬅️ Left Arrow — Move Left
- ➡️ Right Arrow — Move Right
